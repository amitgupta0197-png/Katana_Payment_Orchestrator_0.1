Platform

The whole stack, from
button to settled rupee.

One platform for collection, reconciliation, routing, and settlement — behind one signed API, in any language.

Hosted checkout, any language

POST a signed order, redirect to a hosted pay page. PHP, Node, Python, Go, WordPress — if it can send JSON, it can take payments.

Real-time reconciliation

Every UPI credit is captured, matched to its order, and reconciled — with RRN/UTR tracking so nothing slips through.

Smart multi-provider routing

Each payment scored across providers on success rate, latency, health and cost — routed to the best rail, live.

On-device RRN capture

When banks expose no API, our agent reads the 12-digit UPI reference on-device so collections always reconcile.

Fast branch settlements

A full upline → Katana → branch settlement flow with immutable timelines, versioned commission rules, and USDT or bank payouts.

Developer-first API

Signed order API, hosted pay page, signed callbacks. Per-branch Key + Salt, live OpenAPI docs, copy-paste samples.
